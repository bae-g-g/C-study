# Relational

## **Relational Operator 관계 연산자**

대소 비교

| \== | 좌항값과 우항값이 같은지 확인 | A == B |
| ---| ---| --- |
| != | 좌항값과 우항값이 다른지 확인 | A != B |
| \> | 좌항값이 더 큰지 확인 | A > B |
| < | 우항값이 더 큰지 확인 | A < B |
| \>= | 좌항값이 크거나 같은지 확인 | A >= B |
| <= | 우항값이 크거나 같은지 확인 | A <= B |

False = 0
True = 1

※ 조건식: False = 0, True = 1
※ 조건: False = 0, True = 0이아닌 나머지

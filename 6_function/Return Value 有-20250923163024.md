# Return Value 有

## **반환값(Reutrn값) '有' 함수**
###   

**c\_function\_1.c**

```cpp
#include <stdio.h>

int function_1()     // 함수 선언 '동시' 정의:  반환 자료형 int, 함수명 function_1, 매개변수 0개 함수
{
    return 99;       // 반환값 99 정수 = 3번째줄 반환자료형 int
}

int main()
{
    int result = function_1();               
    printf("함수 function_1 실행종료\n");
    printf("반환값: %d\n", result);

    return 0;
}
```

※ Compiling
① 메인 함수 내 int result = function\_1(); 실행
② 사용자정의 함수 function\_1(); 호출
③ 메인 함수 변수 초기화값(인수) 사용자정의 함수 매개변수에 재초기화 → 없음
사용자정의 함수 매개변수 0개 = 메인 함수 변수 초기화값(인수) 無
④ 사용자정의 함수 반환값(Return값) 정수 99 반환(Return)
⑤ 메인 함수내 function\_1()을 사용자정의 함수 반환값(Return값)대체
function\_1() = 99
⑥ 99를 result 변수에 초기화 (int rult = 99)
⑦ printf 실행 → result 변수기화값 99 출력
⑧ 메인 함수 반환값(Return값) 0 반환 → 프로그램 정상종료

2-2 조건문
사용자정의 함수 반환값 1 (개발자 內 약속) → 정상종료 → 조건문 조건/조건식 True
사용자정의 함수 반환값 0 (개발자 內 약속) → 비정상종료 → 조건문 조건/조건식 False

#include <stdio.h>

{
    if (check_score >= 60)
    {
        return 1;

    else
    {
        return 0;
